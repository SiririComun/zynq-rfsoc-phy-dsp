\section{Introduction}
This monograph examines the SoC bridge between processing system and programmable logic, focusing on control and data-plane interactions across AXI protocol families. It sets the foundation for a consistent memory-map methodology that aligns low-rate register control with high-rate \AXIStream transport.

The narrative highlights integration constraints that govern reliable software-hardware exchange for \IandQ pipelines while maintaining deterministic behavior under practical bandwidth and latency budgets.
