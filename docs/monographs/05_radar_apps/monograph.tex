\section{Introduction}
This monograph consolidates the complete radar application pathway from waveform definition through hardware execution to software-level interpretation. It positions the prior monographs as composable layers in a full-stack sensing workflow suitable for thesis-grade experimentation.

The focus is on demonstrating how architectural choices propagate to application outcomes, including detectability, robustness, and quantitative limits linked to \SNR, \ENOB, and calibrated \dBFS measurement practices.
